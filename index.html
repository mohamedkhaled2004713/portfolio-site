GRADUATION PROJECT

Smart Glasses

AI-Powered Wearable for the Visually Impaired

An innovative wearable AI device that transforms how visually impaired individuals interact with the world. By integrating real-time Computer Vision, OCR, Face Recognition, and Voice AI into a single Python-powered system, the Smart Glasses provide essential auditory feedback — from reading text aloud to recognizing faces and detecting obstacles — making daily life safer and more accessible.

Core Features

Object Detection

Real-time YOLOv8 object detection with directional audio feedback — tells user what's on their left, right, or ahead.

OCR Text Reading

Reads any text in front of the camera powered by Gemini AI — from documents to street signs and product labels.

Face Recognition

Learns and remembers faces. Announces known people by name with 25-frame training for high accuracy.

Safety Mode

Continuous obstacle monitoring with proximity warnings — alerts the user when objects get dangerously close.

Currency Recognition

Identifies banknotes and their value using AI vision — helping users handle money independently.

Emergency SOS

Voice-activated distress signal that sends email with GPS location to emergency contacts instantly.

Voice Control

Full Arabic voice command system — switch modes, save faces, ask time, and trigger features hands-free.

Battery Saver Mode

Intelligent power management that disables heavy AI tasks to extend battery life when needed.
