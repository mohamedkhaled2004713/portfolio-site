CYBERSECURITY SYSTEM

Sentinel-AI

Multi-Agent Fraud Detection & Cybersecurity Hub

Sentinel-AI is an intelligent multi-agent system designed to analyze transactions, detect threats, and make real-time security decisions. It combines three specialized AI agents working in harmony to provide comprehensive fraud detection and cybersecurity analysis.

Core Agents

Analyzer Agent

Reads transaction data and measures anomaly scores with rapid precision. Detects unusual patterns and calculates risk indicators in real-time.

Forensics Agent

Traces technical paths and analyzes exploitation indicators. Investigates transaction routes and identifies suspicious behavioral patterns.

Decision Agent

Issues final verdict: SAFE, SUSPICIOUS, or FRAUD. Provides recommendations and confidence scores based on all agent analyses.

Real-Time Processing

Analyzes transactions instantly with sub-second latency. Provides immediate alerts and actionable insights for security teams.

GRADUATION PROJECT

Smart Glasses

AI-Powered Wearable for the Visually Impaired

An innovative wearable AI device that transforms how visually impaired individuals interact with the world. By integrating real-time Computer Vision, OCR, Face Recognition, and Voice AI into a single Python-powered system, the Smart Glasses provide essential auditory feedback — from reading text aloud to recognizing faces and detecting obstacles — making daily life safer and more accessible.

Core Features

Object Detection

Real-time YOLOv8 object detection with directional audio feedback — tells user what's on their left, right, or ahead.

OCR Text Reading

Reads any text in front of the camera powered by Gemini AI — from documents to street signs and product labels.

Face Recognition

Learns and remembers faces. Announces known people by name with 25-frame training for high accuracy.

Safety Mode

Continuous obstacle monitoring with proximity warnings — alerts the user when objects get dangerously close.

Currency Recognition

Identifies banknotes and their value using AI vision — helping users handle money independently.

Emergency SOS

Voice-activated distress signal that sends email with GPS location to emergency contacts instantly.

Voice Control

Full Arabic voice command system — switch modes, save faces, ask time, and trigger features hands-free.

Battery Saver Mode

Intelligent power management that disables heavy AI tasks to extend battery life when needed.

IMAGE PROCESSING APP

PixelForge

Real-Time Image Filtering & Processing Studio

PixelForge is a powerful, professional real-time image filtering web application built with pure HTML5, CSS3, and Vanilla JavaScript using the Canvas API. It delivers pixel-level image manipulation with zero dependencies — drag, drop, filter, and export in seconds.

Core Features

Smart Upload

Drag & Drop, file browser, or paste directly from clipboard (Ctrl+V). Supports JPG, PNG, WEBP & GIF up to 20MB.

20+ Filters

Grayscale, Edge Detection, Blur, Sepia, Invert, Emboss, Sharpen, Vintage, Cartoon, Oil Paint and many more effects.

Stack Filters

Enable Stack Mode to combine multiple filters simultaneously for unique, creative effects.

8 One-Click Presets

Cinematic, Noir, Dreamy, Retro, Comic, Ghost, Cyberpunk & Watercolor — instant professional looks.

3 View Modes

Preview, Side-by-Side Compare, and interactive Split Slider — see your edits with full context.

Live Histogram

Real-time RGB histogram displayed during filtering. Brightness, Contrast, Blur & Intensity controls.

Multi-Format Export

Export in PNG (lossless), JPEG (adjustable quality), or WebP (modern format) with one click.

Zero Dependencies

Runs entirely in the browser — no server, no libraries, no install. Just open index.html and start creating.

Tech Stack
